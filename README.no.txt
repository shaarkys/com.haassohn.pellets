Administrer din Haas+Sohn pelletovn fra Homey via ditt lokale nettverk. Appen viser sanntidsstatus som aktuell og ønsket temperatur, driftsmodus, eco- eller ukeprogram, pelletforbruk samt rengjørings- eller vedlikeholdsindikatorer, og den kan varsle om feilkoder.

Med Homey kan du slå ovnen av eller på, sette ønsket temperatur, slå eco-modus eller ukeprogram av/på og følge med på pelletnivået. For å komme i gang, skriv inn ovnens IP-adresse eller vertsnavn og PIN-koden som er konfigurert på kontrolleren. Inspirert av https://github.com/ioBroker/ioBroker.hass.
