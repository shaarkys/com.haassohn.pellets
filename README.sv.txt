Hantera din Haas+Sohn pelletskamin från Homey via ditt lokala nätverk. Appen visar live-status som aktuell och önskad temperatur, driftläge, eco- eller veckoprogram, pelletförbrukning samt rengörings- eller underhållsindikatorer, och kan varna för felkoder.

Med Homey kan du slå på eller av kaminen, ställa in önskad temperatur, växla eco-läge eller veckoprogram och hålla koll på pelletnivån. För att komma igång, ange kaminens IP-adress eller värdnamn samt PIN-koden som är konfigurerad på kontrollern. Inspirerad av https://github.com/ioBroker/ioBroker.hass.
